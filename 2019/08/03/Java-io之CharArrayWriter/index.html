write(char[], int, int)的参数校验

这样严谨的参数校验值得学习 可以在日后的代码中用起来

  • off < 0: 起始位置必须大于等于0
  • off > c.length: 起始位置必须小于数组长度
  • len: 读取的数据数量必须 >= 0
  • off + len) > c.length: 读取的最后一个数据位置必须不能超过数组长度
  • off + len) < 0: off与len相加不应溢出int
